## Development

Make sure you have [Node.js](https://nodejs.org) and [PNPM](https://pnpm.io) installed.

### Install Dependencies

```bash
pnpm install
```

### Development Server

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
